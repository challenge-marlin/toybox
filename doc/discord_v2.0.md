# 🎉 TOYBOX! Ver2.0 大型アップデートリリース！

いつも TOYBOX! を使ってくれてありがとう！
今日から **Ver2.0** が正式リリースされました！

> **「もっと投稿したくなる・毎日来たくなる」** を目標にした大型アップデートです 🚀

---

## 👍 リアクションが6種類に！

これまで「いいね」だけだったリアクションが **6種類** に増えました！
気持ちにぴったりなものを選んで、作品に感想を送ってあげてね💬

| 絵文字 | 名前 |
|--------|------|
| 👍 | いいね！ |
| 🤩 | すごい！ |
| 🥰 | かわいい！ |
| 😂 | 笑える！ |
| 😭 | 感動した |
| 😎 | かっこいい！ |

投稿を **タップして拡大したときもリアクションできる** ようになったよ！

---

## 🏅 称号システムが全面リニューアル！

### ✅ 有効期限が廃止
もう「1週間で消える」はなし。**一度もらった称号は永久保有** できます！

### ✅ 達成条件が明確に
投稿数・ゲーム投稿・もらったいいね数など、**何をすれば何の称号が解放されるか** 一覧で確認できます。

### ✅ 称号が約50種類に拡充！
難易度ごとにカラーが違うバッジ形式でコレクション可能🎖️
中には **存在すら秘密の超高難易度称号** も…！

### ✅ 称号コレクション画面が新設
マイページ →「**獲得称号**」から全称号を一覧チェック！
- 取得済み → フルカラーで点灯 ✨
- 未取得・条件公開 → グレーアウト＋残り条件表示
- 秘密称号 → **「？？？」** 達成したときに初めて名前が判明！

---

## ✨ TOYBOXポイント（TP）システム新登場！

活動するとポイントが貯まる新システムが始まりました！

| アクション | ポイント |
|------------|---------|
| 🔑 毎日ログイン | 30 TP |
| 🖼️ 画像を投稿 | 10 TP |
| 🎬 動画を投稿 | 25 TP |
| 🎮 ゲームを投稿 | 100 TP |
| 💌 リアクションをもらう | 3〜5 TP |

貯まったTPは **マイページ右上** で確認できます（自分のみ表示）。

### 🎁 移行ボーナス：全員に100TP プレゼント！

Ver2.0リリースを記念して、**すでにアカウントをお持ちの全ユーザーに 100 TP** を配布しました！
マイページを開いて確認してみてね👀

## 🕹️ 「AIでゲームをつくる」ボタン追加
