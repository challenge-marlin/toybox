# リストアクイックリファレンス

> 経験者向けの簡潔なコマンド集

## サーバー間リストア（本番→ステージング）

### 本番サーバーでバックアップ作成

```bash
cd /var/www/toybox/backend
sudo docker compose exec -T db pg_dump -U postgres -F c toybox > /backup/toybox/database/toybox_$(date +%Y%m%d_%H%M%S).dump
sudo /tmp/backup_media_from_prod.sh
```

### ステージングサーバーでリストア

```bash
# ファイルをコピー（本番サーバーから）
scp root@本番IP:/backup/toybox/database/toybox_YYYYMMDD_HHMMSS.dump /tmp/
scp root@本番IP:/backup/toybox/media/media_volume_YYYYMMDD_HHMMSS.tar.gz /tmp/

# データベースリストア
cd /var/www/toybox/backend
sudo docker compose exec db psql -U postgres -c "DROP DATABASE IF EXISTS toybox; CREATE DATABASE toybox;"
sudo docker cp /tmp/toybox_YYYYMMDD_HHMMSS.dump backend-db-1:/tmp/restore.dump
sudo docker compose exec db pg_restore -U postgres -d toybox -v /tmp/restore.dump
sudo docker compose exec web python manage.py migrate
sudo docker compose restart web

# メディアリストア
sudo docker run --rm -v backend_media_volume:/data alpine rm -rf /data/*
sudo docker run --rm \
  -v backend_media_volume:/target \
  -v /tmp/media_volume_YYYYMMDD_HHMMSS.tar.gz:/backup.tar.gz:ro \
  alpine tar xzf /backup.tar.gz -C /target
```

## サーバー→ローカルリストア

### WinSCPでダウンロード

```
/backup/toybox/database/toybox_YYYYMMDD_HHMMSS.dump
→ C:\backup\toybox-verification\database\

/backup/toybox/media/media_volume_YYYYMMDD_HHMMSS.tar.gz
→ C:\backup\toybox-verification\volumes\
```

### ローカルでリストア（PowerShell）

```powershell
cd C:\github\toybox\backend

# データベース
docker compose exec db psql -U postgres -c "DROP DATABASE IF EXISTS toybox; CREATE DATABASE toybox;"
docker cp "C:\backup\toybox-verification\database\toybox_YYYYMMDD_HHMMSS.dump" backend-db-1:/tmp/restore.dump
docker compose exec db pg_restore -U postgres -d toybox -v /tmp/restore.dump
docker compose exec web python manage.py migrate
docker compose restart web

# メディア
docker run --rm -v backend_media_volume:/data alpine rm -rf /data/*
$BACKUP_FILE = "C:\backup\toybox-verification\volumes\media_volume_YYYYMMDD_HHMMSS.tar.gz"
docker run --rm -v backend_media_volume:/target -v "${BACKUP_FILE}:/backup.tar.gz:ro" alpine tar xzf /backup.tar.gz -C /target

# URL修正（ローカルのみ）
docker compose exec db psql -U postgres -d toybox -c "UPDATE users SET avatar_url = REPLACE(avatar_url, 'https://toybox.ayatori-inc.co.jp', 'http://localhost:8000');"
docker compose exec db psql -U postgres -d toybox -c "UPDATE user_meta SET header_url = REPLACE(header_url, 'https://toybox.ayatori-inc.co.jp', 'http://localhost:8000');"
docker compose exec db psql -U postgres -d toybox -c "UPDATE submissions SET image_url = REPLACE(image_url, 'https://toybox.ayatori-inc.co.jp', 'http://localhost:8000'), video_url = REPLACE(video_url, 'https://toybox.ayatori-inc.co.jp', 'http://localhost:8000'), game_url = REPLACE(game_url, 'https://toybox.ayatori-inc.co.jp', 'http://localhost:8000');"
```

## トラブルシューティング

### Dockerコンテナが起動していない

```bash
# サーバー
cd /var/www/toybox/backend
sudo docker compose up -d

# ローカル
cd C:\github\toybox\backend
docker compose up -d
```

### セッションエラー

```powershell
docker compose exec db psql -U postgres -d toybox -c "DELETE FROM django_session;"
docker compose restart web
```

### 画像が表示されない

```powershell
# シンボリックリンク作成（ローカル）
docker compose exec web sh -c "rm -rf /app/media && ln -s /app/public/uploads /app/media"
docker compose restart web
```

## 確認コマンド

```bash
# データベース接続確認
docker compose exec db psql -U postgres -d toybox -c "SELECT COUNT(*) FROM users;"

# メディアボリューム確認
docker run --rm -v backend_media_volume:/data alpine ls -la /data/

# コンテナ状態確認
docker compose ps
```
