# リストア実行手順（最終版）

**作成日**: 2026年1月29日  
**バックアップディレクトリ**: `C:\backup\toybox-verification\database`

---

## ステップ1: ダウンロードしたファイルを確認

```powershell
Get-ChildItem C:\backup\toybox-verification\database\
```

このコマンドを実行して、ファイル名を確認してください。

---

## ステップ2: ファイルを解凍（エクスプローラーで実行）

1. エクスプローラーで `C:\backup\toybox-verification\database\` を開く
2. `toybox_*.sql.gz` ファイルを右クリック
3. 「7-Zip」→「ここに解凍」を選択
4. `toybox_*.sql` ファイルが作成されることを確認

---

## ステップ3: 解凍したファイルを確認

```powershell
Get-ChildItem C:\backup\toybox-verification\database\*.sql
```

表示されたファイル名をメモしてください（例: `toybox_20260129_133225.sql`）

---

## ステップ4: データベースをリセット

```powershell
cd C:\github\toybox\backend
docker compose exec db psql -U postgres -c "DROP DATABASE IF EXISTS toybox;"
docker compose exec db psql -U postgres -c "CREATE DATABASE toybox WITH ENCODING='UTF8' LC_COLLATE='C' LC_CTYPE='C' TEMPLATE=template0;"
```

---

## ステップ5: リストア実行

**以下のコマンドで、ファイル名を実際のファイル名に置き換えてから実行してください：**

```powershell
# ファイル名を実際のファイル名に置き換える
$sqlFile = "C:\backup\toybox-verification\database\toybox_20260129_133225.sql"

# リストア実行
$content = Get-Content $sqlFile -Encoding UTF8 -Raw
$content | docker compose exec -e PGCLIENTENCODING=UTF8 -e LANG=C.UTF-8 -T db psql -U postgres toybox
```

**重要**: `toybox_20260129_133225.sql` を、ステップ3で確認したファイル名に置き換えてください。

---

## ステップ6: リストア結果を確認

```powershell
# 日本語が正しく表示されるか確認
docker compose exec db psql -U postgres -d toybox -c "SELECT id, name FROM gamification_title LIMIT 5;"

# ユーザー情報を確認
docker compose exec db psql -U postgres -d toybox -c "SELECT id, username, display_name FROM users LIMIT 5;"

# データの件数を確認
docker compose exec db psql -U postgres -d toybox -c "SELECT COUNT(*) FROM users;"
```

**期待される結果**: 日本語が正しく表示される（`?????`ではない）

---

## ステップ7: メディアボリューム（画像ファイル）をリストア

### 7-1. メディアボリュームバックアップを確認

```powershell
Get-ChildItem C:\backup\toybox-verification\volumes\*.tar.gz
```

### 7-2. リストア実行

```powershell
# ファイル名を確認してから、実際のファイル名に置き換える
$mediaFile = "media_volume_20260129.tar.gz"

# Dockerボリューム名を確認
docker volume ls | Select-String "media"

# リストア実行
docker run --rm -v backend_media_volume:/data -v "C:\backup\toybox-verification\volumes:/backup" alpine sh -c "cd / && tar xzf /backup/$mediaFile"
```

**重要**: 
- `media_volume_20260129.tar.gz` を実際のファイル名に置き換える
- `backend_media_volume` をDockerボリューム名に置き換える（必要に応じて）
