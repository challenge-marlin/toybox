# ドキュメント

このディレクトリには、プロジェクトのドキュメントが整理されています。

## 構成

- **`legacy/`** - Django移行前のコード（参考用）
  - `backend/` - Express + TypeScript + MongoDB のバックエンドコード
  - `frontend/` - Next.js のフロントエンドコード

- **`deployment/`** - デプロイメント関連ドキュメント
  - `DEPLOYMENT_GUIDE.md` - デプロイメントガイド
  - `DEPLOYMENT_SUMMARY.md` - デプロイメントサマリー
  - その他のデプロイメント関連ファイル

- **`setup/`** - セットアップ関連ドキュメント
  - `SETUP_INSTRUCTIONS.md` - セットアップ手順
  - `SETUP_DEPLOYMENT.md` - デプロイメントセットアップ
  - セットアップスクリプト（`.sh`ファイル）

- **`troubleshooting/`** - トラブルシューティング
  - `TROUBLESHOOTING_NGINX.md` - Nginx関連のトラブルシューティング
  - `BUGFIX_SUMMARY.md` - バグ修正サマリー
  - その他のトラブルシューティング関連ファイル

- **`migration/`** - 移行ドキュメント（`docs/migration/`を参照）
