# Webブラウザゲームの作り方

## 概要（何ができる？）

AI（ChatGPT、Claude、GitHub Copilotなど）を使って、Webブラウザで動作するミニゲームを作成できます。TOYBOXでは、作成したゲームをZIPファイルでアップロードして投稿できます。

## 事前準備（必要なもの）

1. **AIツールのアカウント**
   - 推奨：ChatGPT（GPT-4）、Claude、GitHub Copilot
   - 無料プランでも利用可能なサービスもあります

2. **基本的な知識**
   - HTML、CSS、JavaScriptの基礎（AIが生成してくれるので、理解できればOK）
   - ファイルの圧縮（ZIP形式）

3. **テキストエディタ**
   - Visual Studio Code、Sublime Text、メモ帳など

## 手順

### ステップ1: AIツールを選ぶ

人気のツール：
- **ChatGPT (GPT-4)**: コード生成が得意。会話形式で修正可能
- **Claude**: 長いコードも生成可能。説明が丁寧
- **GitHub Copilot**: エディタ内で直接コード生成

### ステップ2: ゲームの仕様を決める

TOYBOXのお題ジェネレーターで「ゲーム」ボタンを押して、お題を取得しましょう。

仕様プロンプトの例：
```
HTML5で動作するシンプルなミニゲームを作成してください。
- ゲームタイプ: パズルゲーム
- 操作方法: マウスクリック
- 目標: ブロックを揃える
- スタイル: カラフルで楽しいデザイン
```

### ステップ3: AIにプロンプトを送る

1. 選んだAIツールにログイン
2. お題と仕様を組み合わせてプロンプトを作成
3. AIに送信してコードを生成してもらう

プロンプト例：
```
TOYBOXのお題: 「ブロックパズルゲーム」
以下の要件でHTML5ゲームを作成してください：
- index.html 1ファイルで完結
- カラフルなブロックをマウスで操作
- スコア機能付き
- モダンなデザイン
```

### ステップ4: 生成されたコードを確認する

- AIが生成したHTMLコードをコピー
- コードに `index.html` という名前を付けて保存
- ブラウザで開いて動作確認

### ステップ5: 必要に応じて修正する

- 動作しない場合は、AIに「エラーが出ます。修正してください」と伝える
- デザインを変更したい場合は、「色を変更してください」などと指示

### ステップ6: ZIPファイルを作成する

1. `index.html` ファイルを1つだけ含むZIPファイルを作成
2. ファイル名は任意（例：`my-game.zip`）
3. サイズ：最大50MBまで

**重要**: ZIPファイルには必ず `index.html` が含まれている必要があります。

### ステップ7: TOYBOXに投稿する

1. マイページの「ゲームをアップロード（ZIP）」からZIPファイルを選択
2. タイトルとキャプションを入力
3. ハッシュタグを追加（任意）
4. サムネイル画像を設定（任意）
5. 同意チェックボックスにチェック
6. 「投稿する」ボタンをクリック

## よくある失敗と対策

### 失敗1: ゲームがブラウザで動かない

**対策**: 
- ブラウザの開発者ツール（F12）でエラーを確認
- AIに「ブラウザで動作確認してください」と伝えて修正を依頼
- `index.html` のファイル名が正しいか確認

### 失敗2: ZIPファイルにindex.htmlが含まれていない

**対策**: 
- ZIPファイルを展開して、`index.html` が含まれているか確認
- ZIPファイルの作成方法を再確認（フォルダごと圧縮ではなく、ファイルを直接圧縮）

### 失敗3: ゲームが複雑すぎて動作が重い

**対策**: 
- AIに「もっとシンプルなゲームにしてください」と伝える
- 不要な機能を削除
- 画像やアニメーションを減らす

## 次にやること（TOYBOXに投稿するまで）

1. ✅ お題を取得（お題ジェネレーター）
2. ✅ ゲームの仕様を決める
3. ✅ AIにプロンプトを送ってコードを生成
4. ✅ ブラウザで動作確認
5. ✅ 必要に応じて修正
6. ✅ ZIPファイルを作成
7. ✅ TOYBOXにアップロード
8. ✅ タイトル・キャプション・ハッシュタグを入力
9. ✅ 投稿！

投稿後は、他のユーザーがあなたのゲームをプレイして楽しんでくれます！
