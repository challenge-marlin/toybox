# 簡単リストア手順（コピー&ペーストで実行）

**作成日**: 2026年1月29日  
**目的**: 確実にリストアを完了する

---

## 重要な注意事項

- **各コマンドを1行ずつコピー&ペーストして実行してください**
- **エラーが出たら、そのエラーメッセージを確認してから次に進んでください**

---

## ステップ1: ダウンロードしたファイルを確認

```powershell
# C:\backup\toybox\database\ にあるファイルを確認
Get-ChildItem C:\backup\toybox\database\
```

**期待される結果**: ファイル一覧が表示される

---

## ステップ2: データベースをリセット

```powershell
cd C:\github\toybox\backend
docker compose exec db psql -U postgres -c "DROP DATABASE IF EXISTS toybox;"
docker compose exec db psql -U postgres -c "CREATE DATABASE toybox WITH ENCODING='UTF8' LC_COLLATE='C' LC_CTYPE='C' TEMPLATE=template0;"
```

---

## ステップ3: リストア（ファイル名を手動で確認してから実行）

### 3-1. ファイルを手動で解凍

1. エクスプローラーで `C:\backup\toybox\database\` を開く
2. `toybox_*.sql.gz` ファイルを右クリック
3. 「7-Zip」→「ここに解凍」を選択
4. `toybox_*.sql` ファイルが作成されることを確認
5. ファイル名をメモ（例: `toybox_20260129_133225.sql`）

### 3-2. リストア実行

**以下のコマンドで、ファイル名を実際のファイル名に置き換えてから実行してください：**

```powershell
# ファイル名を確認
Get-ChildItem C:\backup\toybox\database\*.sql

# リストア実行（ファイル名を置き換える）
$sqlFile = "C:\backup\toybox\database\toybox_20260129_133225.sql"
$content = Get-Content $sqlFile -Encoding UTF8 -Raw
$content | docker compose exec -e PGCLIENTENCODING=UTF8 -e LANG=C.UTF-8 -T db psql -U postgres toybox
```

**重要**: `toybox_20260129_133225.sql` を実際のファイル名に置き換えてください。

## ステップ4: リストア結果を確認

```powershell
# 日本語が正しく表示されるか確認
docker compose exec db psql -U postgres -d toybox -c "SELECT id, name FROM gamification_title LIMIT 5;"

# ユーザー情報を確認
docker compose exec db psql -U postgres -d toybox -c "SELECT id, username, display_name FROM users LIMIT 5;"
```

**期待される結果**: 日本語が正しく表示される（`?????`ではない）

---

## ステップ5: メディアボリューム（画像ファイル）をリストア

### 5-1. ファイルを確認

```powershell
# メディアボリュームバックアップを確認
Get-ChildItem C:\backup\toybox\volumes\*.tar.gz
```

### 5-2. リストア実行（ファイル名を置き換える）

```powershell
# ファイル名を確認してから実行
$mediaFile = "media_volume_20260129.tar.gz"
docker run --rm -v backend_media_volume:/data -v "C:\backup\toybox\volumes:/backup" alpine sh -c "cd / && tar xzf /backup/$mediaFile"
```

**重要**: `media_volume_20260129.tar.gz` を実際のファイル名に置き換えてください。

## トラブルシューティング

### 問題1: ファイルが見つからない

```powershell
# 存在するファイルを確認
Get-ChildItem C:\backup\toybox\database\
Get-ChildItem C:\backup\toybox\volumes\
```

### 問題2: 7-Zipがインストールされていない

- 7-Zipをダウンロードしてインストール: https://www.7-zip.org/

### 問題3: それでも文字化けが発生する

サーバー上で新しいバックアップを取得してから、WinSCPでバイナリモードでダウンロードしてください。
詳細は `ダウンロード時の文字化け対策.md` を参照。
