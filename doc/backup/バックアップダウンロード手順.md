# バックアップファイルのダウンロード手順

**作成日**: 2026年1月29日  
**目的**: 本番サーバーからバックアップファイルをローカルにダウンロードする

---

## 前提条件

- SSH鍵が存在すること: `C:\Users\ayato\.ssh\toybox-2025-11-06-11-40.pem`
- 本番サーバーにアクセスできること: `root@160.251.168.144`

---

## ダウンロード手順

### ステップ1: バックアップディレクトリを作成

```powershell
# PowerShellで実行
New-Item -ItemType Directory -Path "C:\backup\toybox\database" -Force
New-Item -ItemType Directory -Path "C:\backup\toybox\volumes" -Force
```

### ステップ2: SQLデータベースバックアップをダウンロード

```powershell
# PowerShellで実行
scp -i "C:\Users\ayato\.ssh\toybox-2025-11-06-11-40.pem" root@160.251.168.144:/backup/toybox/database/toybox_*.sql.gz C:\backup\toybox\database\
```

### ステップ3: メディアボリュームバックアップをダウンロード（画像ファイル）

```powershell
# PowerShellで実行
scp -i "C:\Users\ayato\.ssh\toybox-2025-11-06-11-40.pem" root@160.251.168.144:/backup/toybox/volumes/media_volume_*.tar.gz C:\backup\toybox\volumes\
```

### ステップ4: ダウンロードしたファイルを確認

```powershell
# PowerShellで実行
Get-ChildItem C:\backup\toybox\database\
Get-ChildItem C:\backup\toybox\volumes\
```

---

## 代替方法: WinSCPを使用

1. WinSCPを起動
2. サーバーに接続
   - ホスト: `160.251.168.144`
   - ユーザー名: `root`
   - 秘密鍵: `C:\Users\ayato\.ssh\toybox-2025-11-06-11-40.pem`
3. サーバー側のディレクトリに移動
   - SQLバックアップ: `/backup/toybox/database/`
   - メディアボリューム: `/backup/toybox/volumes/`
4. 最新のファイルをダウンロード
   - `toybox_*.sql.gz` → `C:\backup\toybox\database\`
   - `media_volume_*.tar.gz` → `C:\backup\toybox\volumes\`
